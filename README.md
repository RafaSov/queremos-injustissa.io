# Queremos Injustiça

Guild **QUEREMOS INJUSTIÇA** convida players para guild e core.

Somos um grupo de amigos buscando reviver a glória de se divertir jogando online!

Hoje em maior parte dos jogos a competitividade se tornou a norma, e muitas vezes existe stress e intriga onde deveria existir diversão e amizade. Caso esteja procurando um ambiente que respeita seu tempo, onde a vida real é mais importante que o joguinho online, será bem vindo!

Estamos procurando players novos e veteranos, estamos dispostos a ensinar e ajudar a evoluir o personagem e o mais importante, a se divertir no processo. Nosso foco é matar boss, fazer dg, equipar, rir e se descontrair. AOTC/KSM/KSH e afins vem de brinde nesse processo.

Se o seu foco é jogar pra se divertir, raidar 1 dia na semana, fazer dgs/delves em equipe, aprender junto sobre o jogo (mecânicas, classe, rotação) e fazer amigos no processo, entre em contato com **raffinhaup** no Discord, ou **raffinhaup#1665** no Battle.net.

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Compilar TypeScript
npm run build

# Servir localmente (http://localhost:3000)
npm run serve
```

## Deploy (GitHub Pages)

O formulário de contato usa [Formsubmit.co](https://formsubmit.co) para enviar mensagens ao e-mail **rafiajogos@gmail.com**. Na primeira submissão, um e-mail de ativação será enviado — basta clicar no link para ativar.

1. Faça push do código para o repositório
2. Em **Settings → Pages**, configure o source para a branch principal
3. O site estará disponível em `https://seu-usuario.github.io/queremos-injustissa.io/`

## Estrutura

- `index.html` — Estrutura da página (Hero, Quem somos, Objetivo, Contato)
- `styles.css` — Estilos e animações
- `main.ts` / `main.js` — Animações de scroll, partículas, parallax
